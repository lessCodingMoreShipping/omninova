# OmniNova Project Context

## Project Identity

- **Name:** OmniNova (AI Automation Agency).
- **Core Value:** Moving from "Black Box" AI to "Clear Architecture".
- **Target Audience:** High-end MKB and Enterprise leaders.
- **Languages:** Dual-language support (Dutch/English) via a persistent state toggle.

## Tech Stack

- **Frontend:** React 19, Vite, Tailwind CSS 4.
- **Animation:** Framer Motion.
- **Icons:** Lucide-React.
- **State:** React `useState` for language and simple UI states.

## Content & Tone

- **Tone:** Professional, consultant-led, pragmatic, and trustworthy.
- **Style:** Minimalist, clean, and high-density (Linear.app inspiration).
- **Key Services:** AI Strategy Audits, n8n Automations, Enterprise AI Deployments.

## Deployment & Workflow

- **Environment:** Vite dev server.
- **Hosting:** Vercel (CI/CD via GitHub).
- **Safe Zone:** Bottom-right corner reserved for future chatbot integration.
